Medical Cost Personal Datasets
Insurance Forecast by using Linear Regression

As variveis desse dataset so:

  • age: idade da pessoa.
  • sex: gnero da pessoa.
  • bmi: ndice de massa corporal da pessoa.
  • children: nmero de dependentes da pessoa.
  • smoker: se a pessoa fumante ou no
  • region: regio em que a pessoa mora.
  • charges: custos mdicos individuais cobrados pelo plano de sade

O objetivo primrio nesse dataset fazer a previso da varivel custos. Fazer essa previso fundamental e de interesse de qualquer instituio que venda planos de sade. Fazer essa precificao corretamente fundamental para adquirir uma vantagem competitiva no mercado. Uma boa precificao garante que a instituio est operando dentro dos limites oramentrios e que cada pessoa pague o mais prximo do que seria uma parcela justa, de acordo com suas variveis de entrada.

E os objetivos desse case so:

  • Ilustrar os recursos do PyCaret
  • Criar uma soluo rpida e com boa performance.
  • Explorar os contextos de regresso e classificao.
  • Mostrar que possvel colocar uma nova tica em conjuntos clssicos da literatura.
  • Procedimento end-to-end de Machine Learning: dos dados ao deploy.

Um possve objetivo secundrio seria fazer um modelo para previso da varivel fumante para detectar possveis fraudadores dessa varivel. Como uma varivel que impacta significativamente no preo dos planos de sade, algumas pessoas podem se ver tentadas a responderem que no so fumantes, quando na verdade so. Isso levaria a precificaes incorretas e injustas. Para resolver isso, um modelo pode ser empregado e indicar a probabilidade de um indivdio estar comentendo fraude. Os indivduos com maior probabilidade de fraude podem seguir para uma investigao mais detalhada do caso, de modo a evitar perdas maiores.

  • Anlise Exploratria de Dados (Rpida).

Considerações Finais¶
Veja que o conjunto de dados utilizado nessa análise é um dos clássicos da literatura. Não é um conjunto de dados que poderíamos dizer que representa dados da realidade, pois já estão muito bem resumidos e pré-processados. Esses conjuntos de dados são menos interessantes para se utilizar em projetos de portfólio do que dados que venham de uma fonte mais representativa da realidade.¶
No entanto, uma maneira de tornar o uso de tais conjuntos é tentar colocar uma nova ótica, uma nova abordagem, uma nova maneira de enxergar o problema. É o que nós fizemos ao colocar a atenção da modelagem numa das variáveis que a principio era preditora do problema tradicionalmente abordado, contextualizando com um possível problema de negócio.¶
Isso demonstra sua capacidade de ir além, de explorar novos caminhos, de ser diferente dos demais. É isso que muitos recrutadores estão procurando, então, não perca as chances de demonstrar suas habilidades, em qualquer contexto de aplicação que seja.¶
